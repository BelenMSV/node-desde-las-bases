# 🚀 Curso Completo de Node.js: De Cero a Arquitectura de APIs

¡Bienvenido al curso definitivo para dominar el Backend con JavaScript! 

Este curso está diseñado para desmitificar la "magia" del desarrollo web. No empezamos instalando frameworks que hacen las cosas por ti; empezamos desde los cimientos: interactuando con el sistema operativo, construyendo herramientas de consola (CLIs), y dominando el protocolo HTTP nativo. 

Al finalizar, habrás construido APIs profesionales conectadas a bases de datos **SQL** y **NoSQL**, desplegadas en la nube y protegidas con estándares de la industria.

---

## 📊 Ficha Técnica del Curso

* **Dificultad:** Principiante / Intermedio.
* **⏱️ Tiempo total estimado:** 26 - 35 horas de estudio y práctica.
* **Modalidad:** 100% Práctico (Hands-on).
* **Stack:** Node.js, Express, MongoDB (Mongoose), MySQL (mysql2), Git, Render.

---

## 🗺️ Índice del Curso (8 Módulos)

El curso está dividido en 8 módulos secuenciales. Cada concepto construye sobre el anterior.

### 🧱 [Módulo 01: Los cimientos de Node.js](./01-los-cimientos-de-node-js/README.md)
Instalación profesional (`fnm`) y dominio de ES Modules. Entiende qué es realmente el motor V8.
*Duración: 2 - 3 horas.*

### ⚙️ [Módulo 02: Dominando el Core de Node.js](./02-dominando-el-core-de-node-js/README.md)
Lectura de archivos, información del sistema (RAM, CPU), variables de entorno y sistema de Permisos.
*Duración: 3 - 4 horas.*

### 📦 [Módulo 03: El Ecosistema, NPM y CLIs](./03-el-ecosistema-npm-y-herramientas-cli/README.md)
Gestión de paquetes, dependencias de desarrollo y creación de una herramienta CLI global (clon de `ls`).
*Duración: 3 - 4 horas.*

### 🌐 [Módulo 04: La Web con Node.js (Servidores Nativos)](./04-la-web-con-node/README.md)
Creación de servidores HTTP sin librerías. Procesamiento de JSON, validaciones y el origen de CORS.
*Duración: 4 - 6 horas.*

### 🚀 [Módulo 05: La era de Express.js](./05-la-era-de-express-js/README.md)
Estructuración profesional con el patrón MVC, uso de Middlewares y gestión eficiente de rutas.
*Duración: 4 - 5 horas.*

### 🍃 [Módulo 06: Persistencia NoSQL (MongoDB + Mongoose)](./06-persistencia-de-datos-nosql/README.md)
Del "amnesia" del servidor a la persistencia en la nube. Diseña esquemas de validación estrictos y opera con documentos JSON en MongoDB Atlas.
*Duración: 4 - 5 horas.*

### 🐬 [Módulo 07: Bases de Datos Relacionales (MySQL)](./07-Bases-de-Datos-Relacionales/README.md)
Domina el estándar de la industria financiera. SQL puro, prevención de Inyección SQL, uso de Pools de conexiones y el poder de los `JOINs`.
*Duración: 4 - 5 horas.*

### ☁️ [Módulo 08: Despliegue y Producción](./08-despliegue-y-produccion/README.md)
El cierre de oro. GitHub como puente, despliegue continuo en Render, gestión de base de datos remota y blindaje de seguridad final con CORS.
*Duración: 2 - 3 horas.*
