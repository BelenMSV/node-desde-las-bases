# Módulo 01: Los cimientos de Node.js (El fin de la magia)

Bienvenido al primer módulo del curso. Antes de construir aplicaciones complejas o instalar frameworks mágicos, necesitamos entender cómo funciona Node.js bajo el capó. En este módulo romperemos mitos, configuraremos un entorno profesional y daremos nuestros primeros pasos con el sistema de módulos nativo.

**⏱️ Tiempo estimado de estudio y práctica:** 2 - 3 horas.

## 🎯 Objetivos de Aprendizaje

- Comprender qué es Node.js y cómo el motor V8 permite ejecutar JavaScript fuera del navegador.
- Diferenciar entre el entorno de desarrollo web tradicional y el entorno de ejecución del servidor.
- Entender la arquitectura asíncrona y orientada a eventos de Node.js para operaciones de I/O.
- Instalar Node.js de forma profesional utilizando un gestor de versiones (`fnm`).
- Configurar un proyecto desde cero y dominar el sistema de módulos moderno (ES Modules).

## 📚 Temario Detallado

1. **[¿Qué es Node.js realmente?](./01-que-es-nodejs/README.md)**
   > Desmitifica el concepto de Node.js, descubre el papel del motor V8 y aprende por qué JavaScript ahora vive fuera del navegador.

2. **[Arquitectura, Ecosistema y Casos de Uso](./02-arquitectura-y-casos-de-uso/README.md)**
   > Explora la asincronía, los eventos y los tres grandes pilares de Node: Scripting, herramientas de línea de comandos (CLI) y servidores.

3. **[Instalación de Node.js (La forma correcta)](./03-instalacion/README.md)**
   > Olvida el instalador oficial. Aprende a usar `fnm` para gestionar versiones de forma rápida y profesional, priorizando siempre la estabilidad de las versiones LTS.

4. **[Primer proyecto y el Sistema de Módulos](./04-primer-proyecto-y-modulos/README.md)**
   > Crea tu primer "Hello World", entiende por qué en Node las extensiones `.js` son obligatorias y configura tu `package.json` para usar `import/export`.
